
This folder contains all the relevant files associated with the creation of the project Audio on the Blockchain.

Within the folder named "Black Hole Sun Synthesiser" all necessary files to compile the Black Hole Sun Synthesiser Max/MSP patch are included.

Within the folder "black-hole-sun-contract" are all the files coded in order to deploy the NFT smart contract.

Within the Folder "dApp" are all the files coded in order to deploy the decentralised application used to mint the NFTs
